# 基于springboot+vue的前后端人事管理系统带万字论文和ppt



## 一、介绍


语言: Java 数据库：MySQL 

前端技术 : Vue + ElementUl

后端技术 : SpringBoot + MyBatisPlus

系统角色：管理员、员工

管理员：登录、个人中心、员工管理、部门管理、员工考勤管理、请假申请管理、加班申请管理、员工工资管理、招聘计划管理、员工培训管理、部门培训管理

用户：登录、个人中心、员工考勤管理、请假申请管理、加班申请管理、员工工资管理、招聘计划管理、员工培训管理、部门培训管理、员工详细管理
